Follow these steps to run a new scheduling simulation:

Navigate to Controls: Locate the "Scheduling Controls" section (typically on the sidebar).

Set Parameters:

Schedule Start Date: Select the initial date from which to begin the simulation.

Rotation Cycle Length: Define the total length (e.g., in days or weeks) of one full schedule rotation.

Number of Cycles to Generate: Set how many complete rotations you want the simulation to project into the future.

Run Simulation: Initiate the simulation (e.g., by clicking a "Generate" or "Run" button).

Review Output: Observe the results in the main output area, which will detail the processes below.
